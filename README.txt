--------------------------- README ---------------------------
Language used : Python 3.6.2

Dataset used: Car Evaluation dataset


Compile instructions:
1. Install all the library packages used.
2. The folder contains a python notebook ‘neuralnet.ipynb’. It can be opened using the Jupyter notebook.
3. Run the python notebook. (Jupyter recommended)
4. View 'neuralnet.html' to see a snapshot of the notebook when executed.
